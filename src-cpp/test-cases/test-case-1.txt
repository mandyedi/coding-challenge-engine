Example test file for Coding Challenge Engine.
The TestRUnner only reads two sections:
Between test_start and test_end.
Betwenn solution_start and solution_end.
Note thet the section names are case sensitive.

The following example tests a solution that multiplies every integer with 2.

5
TEST_END
